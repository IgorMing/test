```
src

│   └── profile.jsx
├── index.jsx
├── reducers
│   ├── index.js
│   └── profile.js
└── utils
    └── connector.js
```
Getting started
===

Install dependencies: `yarn install`

Start application on 8080 port: `yarn start`
